# NM_ATTACH_COMPONENTS

## Summary
Specifies what components are attached at the end of a playing a attach anim

## Values
* NAC_LEFT_HAND = 1
* NAC_RIGHT_HAND
* NAC_LEFT_FOOT = 4
* NAC_RIGHT_FOOT = 8
* NAC_LEFT_FOREARM = 16
* NAC_RIGHT_FOREARM = 32
* NAC_LEFT_SHIN = 64
* NAC_RIGHT_SHIN = 128
* NAC_LEFT_THIGH = 256
* NAC_RIGHT_THIGH = 512
