# MISC::SHOOT_SINGLE_BULLET_BETWEEN_COORDS_IGNORE_ENTITY_NEW

## Summary
Fires an instant hit bullet between the two points taking into account an entity to ignore for damage

* **FLOAT** startCoorsX
* **FLOAT** startCoorsY
* **FLOAT** startCoorsZ
* **FLOAT** endCoorsX
* **FLOAT** endCoorsY
* **FLOAT** endCoorsZ
* **INT** damageCaused
* **BOOL** perfectAccuracy = false
* **WEAPON_TYPE** weapon = -1074790547
* **PED_INDEX** ped = null
* **BOOL** createTraceVfx = true
* **BOOL** allowRumble = true
* **FLOAT** initialVelocity = -1
* **ENTITY_INDEX** ignoreEntity = null
* **BOOL** forceCreateNewProjectileObject = false: Flag ensures we create a NEW projectile object and don't use any equipped projectile objects from the firing entity.
* **BOOL** disablePlayerCoverStartAdjustment = false
* **ENTITY_INDEX** targetEntity = null: Parameter will enable homing code if the weapon is a rocket.
* **BOOL** doDeadCheck = true
* **BOOL** freezeProjectileWaitingOnCollision = false
* **BOOL** setIgnoreCollisionEntity = false
* **BOOL** ignoreCollisionResetNoBB = false: To ignore collision until the projectile leaves the BoundingBox of the given IgnoreEntity
