# HUD::END_TEXT_COMMAND_THEFEED_POST_STATS

## Summary
Displays the stats message/icon in the top left of the HUD

## Parameters
* **STRING** titleString: Title of the stats icon.
* **ABILITY_ICON** icon: Icon to use ( See ABILITY_ICON enum above )
* **INT** totalLevel:
The amount in difference of increase/decrease e.g.
-1, 5, -100, 100 Max 100 both ways
* **INT** currentLevel: Current level value 0-100(%)
* **BOOL** isImportant: TRUE: Flash the message.
* **STRING** contactTxD: Contact, texture dictionary string
* **STRING** contactTxN: Contact, texture name

## Returns
* **INT**: A unique id which can be used to manipulate the item while on screen.
